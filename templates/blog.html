Using Tmux to Speed up Productivity in the Linux Shell

February 20th, 2017

Tmux (Terminal Multiplexer) is a great open source utility for managing multiple virtual terminal instances within a single window. Similar to GNU Screen, Tmux supports the creation of persistant terminal sessions that can be detached from without destroying what is running in the shell. Tmux supports easy splitting of terminal windows into multiple panes, a functionality that is somewhat clunky using Screen.
