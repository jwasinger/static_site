Using Tmux to Speed up Productivity in the Linux Shell

Tmux (Terminal Multiplexer) is a great open source utility for managing multiple virtual terminal instances within a single window. Similar to GNU Screen, Tmux supports the creation of persistant terminal sessions that can be detached from without destroying what is running in the shell. Tmux supports easy splitting of terminal windows into multiple panes, a functionality that is somewhat clunky using Screen.

Install Tmux (Apt/Yum)

Tmux is available from EPEL. To install simply run sudo apt-get tmux or sudo yum install tmux depending on your OS.

Configuration

Tmux stores its configuration in file. As a Vim user, I like to add the following configuration: 

bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

This makes navigating Tmux panes similar to navigating Vim panes (another reason Tmux is an awesome tool!).

Usage

Simply invoke tmux to start a new session.

Create a new window:

Ctrl-b + n

Switch between windows:

Ctrl-b + n / Ctrl-b + p

Vertically split the current terminal into two panes:

Ctrl-b + %

Horizontally split the current terminal into two panes:

Ctrl-b + "

Navigate terminal panes:

Ctrl-b + h / Ctrl-b + j / Ctrl-b + k / Ctrl-b + l

Detach from a session

Ctrl-b + d

Re-attach to a session

tmux attach-ses -s session_name

List active sessions

tmux list-ses

Force kill a session by name

tmux kill-ses -s session_name

Create a named session

tmux new-ses -s session_name

Fin

These are only a small subset of the available configuration options that Tmux ships with. Check out the man page for more detailed documentation.

Happy 'muxing!
