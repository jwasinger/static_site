Using Tmux to Speed up Productivity in the Linux Shell

Tmux (Terminal Multiplexer) is a great open source utility for managing multiple terminal instances. Similar to screen (link), Tmux supports the creation of persistance terminals that can be detached from without destroying what is running in the shell. The one feature that makes tmux definitively supperior to Screen is the introduction of multiple terminal panes on a single window (i.e. terminal multiplexing).

Install Tmux (Apt/Yum) 

	sudo apt-get tmux

	sudo yum install tmux

Configuration

Tmux stores its configuration in 

file
. As a Vim user, I like to add the following configuration: 

bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R
This makes navigating Tmux panes similar to navigating Vim panes (another reason Tmux is an awesome tool!).

Usage

Simply invoke 

tmux
to start a new session.

Create a new window: Ctrl-b + n Switch between windows: Ctrl-b + n / Ctrl-b + p Vertically split the current terminal into two panes: Ctrl-b + % Horizontally split the current terminal into two panes: Ctrl-b + " Navigate terminal panes: Ctrl-b + h / Ctrl-b + j / Ctrl-b + k / Ctrl-b + l Detach from a session Ctrl-b + d Re-attach to a session tmux attach-ses -s \{\{session name\}\} List active sessions tmux list-ses Force kill a session by name tmux kill-ses -s \{\{session name\}\}

Go forth and conquer

I have shown how to install Tmux in addition to the core set of commands that are used to manage sessions, windows and panes. These are only a small subset of the available configuration options that Tmux ships with. Check out the man page for more detailed documentation. Happy 'muxing!
